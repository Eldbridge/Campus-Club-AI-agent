# 校园社团AI助手
> 轻量化校园社团智能管理工具 | 渐进式迭代 | 模块化升级

## 项目总览
本项目面向校园社团轻量化管理需求，采用渐进式迭代开发，从基础版逐步演进。
最终规划为三大核心模块：
1.  **对外咨询AI助手**：面向社团成员与新生的智能问答服务
2.  **数据智能管理**：成员信息、活动数据等的自动化管理，统计与分析
3.  **内部工作AI助手**：社团内部事务流程自动化与协作支持等

项目将从 1 模块起步，无复杂依赖，适合快速部署与二次开发。

## 版本结构
本项目采用**独立版本迭代**模式，每个版本均包含专属文档：
- `v1_script/`：基础脚本实现版，轻量简洁、开箱即用，适合快速验证核心功能
- `v2_oop/`：面向对象重构版，模块化解耦、支持功能扩展与长期维护
- 更多版本迭代中...

每个版本目录内均包含：**完整功能说明 + 更新日志
