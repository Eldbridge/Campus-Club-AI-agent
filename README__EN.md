Campus Club AI Assistant
Lightweight Intelligent Management Tool for Campus Clubs | Progressive Iteration | Modular Upgrade
Project Overview

This project is designed for the lightweight management needs of campus clubs, adopting progressive iterative development and evolving gradually from a basic version.

The final plan includes three core modules:

1. External Consultation AI Assistant: Intelligent Q&A service for club members and new students

2. Data Intelligent Management: Automated management, statistics and analysis of member information, activity data, etc.

3. Internal Work AI Assistant: Automation of internal club workflows, collaboration support, etc.

The project will start with Module 1, with no complex dependencies, making it suitable for rapid deployment and secondary development.

Version Structure

This project adopts an independent version iteration model, with each version containing dedicated documentation:

• v1_script/: Basic script version, lightweight and ready to use, ideal for rapid validation of core functions

• v2_oop/: Object-oriented refactored version, modular decoupling, supporting functional expansion and long-term maintenance

• More versions in progress...

Each version directory includes: Complete Feature Specifications + Update Logs
